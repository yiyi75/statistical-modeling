# Case Study: Composing Wellbeing from Survey Items
- Publication link: https://www.frontiersin.org/journals/education/articles/10.3389/feduc.2024.1472125/full
- Full analysis scripts and data: https://osf.io/pzqhk/overview

## Overview
Demonstration of exploratory and confirmatory factor analysis skills applied to well-being assessment. Identifies optimal factor structure for survey-based well-being measures.

## Primary Outcome
- A parsimonious, interpretable latent wellbeing structure
    - Factor 1: Life Satisfaction + Positive Affect
    - Factor 2: Negative Affect
- Validated using CFA
- Suitable for downstream regression and mediation analyses
- Analysis informed student support services and contributed to evidence-based assessment practices.
