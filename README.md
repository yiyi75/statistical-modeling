# Statistical Modeling
This repository contains reproducible implementations of common and advanced statistical models used in data science and behavioral research. 

## Topics Covered
- **Generalized Linear Models** - Survey-weighted regression, count data with overdispersion
- **Multilevel / Hierarchical Models** - Longitudinal analysis, growth modeling
- **Exploratory & Confirmatory Factor Analysis** - Scale validation, measurement models
- **Structural Equation Modeling & Path Analysis** - Mediation, latent variable modeling

## Structure
Each model folder includes:
- **Problem formulation** - Research question and data context
- **R implementation** - Complete code from data cleaning to visualization
- **Results interpretation** - Statistical findings and practical implications
